# Визуализация операций с деревьями
## Инструкция по компиляции
- Данный csproj и sln файл предназначен для испольнения в JetBrains Rider. Если запускаете через Visual Studio - среда разработки должна сформировать свои файлы на основе исходных в репозитории.
- Открыть Visual Sudio Installer и во вкладке "Классические и мобильные приложения" и установить "Разработка с помощью .NET Multi-Platform App UI".
- Далее установить репозиторий и запустить с помощью Visual Studio.
